# WeaCom
**它是为Fabric的开发者们而生，因TeaCon茶话会仅能够展示支持Forge环境的mod，而不支持仅支持Fabric的Mod，为此，WeaCom就建立了**

# WeaCom的意义
**Fabric加载器主打一个字“轻”，而Wea具有“微”的意思，表示微型Mod加载，Com具有“康”的意思，以表示祝福开发者，WeaCom的中文直译就是“微康”，所以也称作微康模组展览会**
